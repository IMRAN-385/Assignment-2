# 🎙️ TechWave – Podcast Website

A modern and responsive podcast landing page built using **HTML5** and **CSS3**. The project follows the provided Figma design while maintaining clean code, responsive layouts, and smooth user experience across all devices.

## 📌 Features

- ✅ Fully responsive navigation bar
- ✅ Hamburger menu for mobile devices
- ✅ Hero section with background image and podcast artwork
- ✅ Gradient call-to-action buttons
- ✅ About section with podcast statistics
- ✅ Why Choose TechWave feature cards
- ✅ Featured episodes with embedded YouTube videos
- ✅ Host profile section
- ✅ Responsive footer
- ✅ Smooth hover animations
- ✅ Mobile-first responsive design
- ✅ Semantic HTML5 structure
- ✅ Clean and organized CSS

## 💻 Installation

Clone the repository:

```bash
git clone https://github.com/your-username/techwave-podcast.git
```

Go to the project folder:

```bash
cd techwave-podcast
```

Open **index.html** in your browser.
